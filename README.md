# FAQ Accordion Card

This project is based on a [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/faq-accordion-card-XlyjD0Oam/hub). 
The goal was to build a FAQ Accordion Card without using any CSS or JS frameworks, focusing on a modular approach and using a comprehensive set of tools and technologies.

## Tech Stack

- SCSS
- JavaScript (OOP Pattern)
- HTML with BEM Methodologies
- JSON (content-driven)
- Gulp (bundling assets)
- Babel (Transpilation the modern JS in to traditional JS)
- Responsive Design
- Vercel (Deployment)

## Getting Started

To set up this project on your local machine, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/durgeshraju/JSFAQAccordion
   ```
2. Navigate to the project folder:
   ```
   cd FAQAccordion
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Run the local development server:
   ```
   npm start
   ```

The project should now be running on `http://localhost:3000`.
